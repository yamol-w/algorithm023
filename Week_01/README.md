学习笔记

栈(Stack)：
	先进后出
	添加 & 删除 时间复杂度皆为 O(1)
	查询时间复杂度为 O(n)

队列(Queue)：
	先进先出
	添加 & 删除 时间复杂度皆为 O(1)
	查询时间复杂度为 O(n)

双端队列(Deque)：
	栈和队列的结合体
	特点：头和尾均可进行元素的出和入操作
	插入 & 删除 时间复杂度皆为 O(1)
	查询时间复杂度为 O(n)

优先队列(Priority Queue):
	是一个接口；或定义的一种抽象数据结构；底层的实现有很多种。
	插入操作时间复杂度为 O(1)
	取出操作时间复杂度为 O(logN)，按照元素的优先级取出（类似于VIP，可以插队，优先处理）
	底层数据结构较为复杂和多样：heap(堆)，bst，treap
